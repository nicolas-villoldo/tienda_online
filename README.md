# Xotic Food & Drinks - E-commerce 
Sistema de tienda online desarrollado para la gestión de pedidos de tragos y comida.

###  Características Técnicas:
* **Backend:** PHP con arquitectura limpia.
* **Base de Datos:** MySQL utilizando **PDO** para mayor seguridad (Prevención de SQL Injection).
* **Transacciones:** Implementación de `beginTransaction` y `commit` para asegurar la integridad en la creación de pedidos y descuento de stock.
* **Frontend:** Diseño responsivo con CSS puro y FontAwesome.
* **Integración:** Preparado para pasarela de pagos con Mercado Pago.
